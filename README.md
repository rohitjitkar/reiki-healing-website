## 🛠️ Tech Stack

- **HTML5** – Semantic, clean, and well-structured markup.
- **CSS3** – Custom styling with animations, flexbox, grid, and glassmorphism.
- **Vanilla JavaScript** – No external libraries; handles scroll reveals, mobile toggles, and form submission.
- **Google Fonts** – *Playfair Display* (serif) + *Inter* (sans-serif) for elegant typography.
- **Formspree** – Backend form handling for consultation requests.
